# NuclearOption-PilotModelReplacer
Mod for Nuclear Option that replaces the pilot model with another character. This mod comes with 3 characters that you can swap the pilot with

!(Screenshots/ScreenshotA.png)

# Installation
To install this mod, make sure you have [BepInEx](https://github.com/BepInEx/BepInEx) installed on your copy of Nuclear Option. 
In [Releases](https://github.com/BruhBoi50/NuclearOption-PilotModelReplacer/releases), Download and extract the .zip file and move the "PilotModelReplacer" folder into the "BepInEx/plugins" folder
After running the game once, open the "BepInEx\config\com.bruhboi.pilotmodelreplacer.cfg" file in a text editor. Here, you can set the value of "AssetBundleFileName" to any of the files inside the "assets" folder of the mod to change the character
